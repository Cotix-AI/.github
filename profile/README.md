### 👋 关于我们 (About Us)

Cotix AI 是一家专注于前沿人工智能技术研发与应用的公司。我们致力于探索和构建可信赖、智能且富有创造力的AI解决方案。我们的工作涵盖了从大型语言模型（LLM）的基础研究、创新算法设计，到特定领域的应用落地，如音乐生成、教育辅助和代码智能等。我们相信，通过开放、协作和持续创新，能够推动AI技术的发展，并为社会创造积极价值。
